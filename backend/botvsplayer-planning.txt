- On create-game:
    - Create owner
    - Create room and put the owner in
    - Map owner.socketId to each room
    - Create bot players
    - Push bot players into the room
    - Start the room
    - Return the payload with actionType: "create-game"

- On leave-game or disconnect:
    - Remove the binding in the map (owner.socketId -> GameRoom)

- On draw-card:
    - If drawPile is empty:
        - Restock then reshuffle the pile from the discardPile, leaving 1 card on top 
    - If their hand has a card that's playable:
        - Throw error to filter
    - Else:
        - Draw 1 card
        - Send back gameState with actionType

- On play-cards:
    - If player's turn:
        - Process the player's hand pattern
            - If incorrect:
                - Throw errors to filters
            - If correct:
                - Process current turn for the player
                - Process next turn for the bot or the player

    - Go into a loop, breaks when some conditions are met (game-end (1 player left), etc...)
        - If game-end:
            - Destroy the room
            - Send back the gameState to the player
            - Breaks the loop
        - If next turn is bot's turn:
            If no playable cards in hand:
                - Set timeout for 1 second
                - Draw a card 
                - Send back gamestate
                - Repeat until there's a playable card agaisnt the top card

            - Set timeout to simulate thinking
            - Do the dfs and get the patterns:
                - Play the longest sequence
            - Process current turn for the current bot
            - Process next turn for the next bot or player
            - Send back gameState to the client
        - If next turn is player's turn:
            - Breaks the loop

- DFS:
    Loop through the hand
    Compare against the top card
        If valid:
            Push that card into currentPattern
            Set as visited
            Start from that card, visit every other cards
                If valid and not visited:
                    Push that card into currentPattern  
                    Set as visited
                    Repeat the same process
                Else (base case):
                    Push the currentPattern into patterns 
                Upon return, if there are no more cards to visit 
                    Remove the current card as visited
        Else:
            Continue onto the next card
